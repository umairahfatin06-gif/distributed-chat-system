# Distributed Real-Time Chat System

A distributed real-time chat application developed using Node.js, Socket.IO, and Redis. This project allows multiple users to communicate simultaneously through real-time WebSocket communication with support for multi-server messaging and dynamic user management.

## Technologies Used
- Node.js
- Socket.IO
- Redis
- HTML
- CSS
- JavaScript
- WebSocket

## Features
- Real-time messaging
- Multiple user support
- Dynamic user management
- Multi-server communication
- Distributed system architecture
- Fault tolerance using Redis
- Interactive web-based interface

## Project Description
This project was developed as part of an academic project to demonstrate the implementation of distributed systems and real-time communication technologies. The application enables users to exchange messages instantly while maintaining synchronization between multiple servers using Redis and Socket.IO.
